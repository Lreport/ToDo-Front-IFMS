# ToDo-Front-IFMS
Atividade - To Do List (Lista de Tarefas)

Implemente uma lista de tarefas em React que contenha:

Layout de sua escolha;
Possibilite adicionar uma nova tarefa;
Editar e excluir uma tarefa (Tentem fazer, depois irei fazer um exemplo com vocês);
Marcar a tarefa como concluída.
Obs: Pensem e anotem quais itens podem se tornar componentes, se atentem ao uso de props e states.
